Merkle Innovation Email Components Library

The purpose of this library is to preserve all components that have been created, customised, and used by the DBG Team over the years.

For each component, a brief description, a live demo, and information on compatibility across various email clients are provided, along with the option to download it for use in your email campaigns without difficulty.
Please refer to the instructions contained in the README file included in the .zip archive.
